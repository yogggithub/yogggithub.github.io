本文展示了博客中所有可用的自定义 shortcodes。

## 1. Notice 提示框

{{% notice tip %}}
这是提示信息。
{{% /notice %}}

{{% notice warning %}}
这是警告信息。
{{% /notice %}}

{{% notice danger %}}
这是危险信息。
{{% /notice %}}

{{% notice info %}}
这是普通信息。
{{% /notice %}}

{{% notice success %}}
这是成功信息。
{{% /notice %}}

## 2. Details 可折叠内容

{{% details title="点击展开" %}}
这里是隐藏的内容。
{{% /details %}}

## 3. Quote 引用框

{{% quote author="鲁迅" source="呐喊" %}}
世上本没有路，走的人多了，也便成了路。
{{% /quote %}}
