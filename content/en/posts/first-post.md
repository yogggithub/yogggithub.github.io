Welcome to my Hugo blog!

## About This Blog

This is a blog created with Hugo and PaperMod theme.

## Features

- Fast static site generation
- Clean and beautiful PaperMod theme
- Markdown writing support
- Responsive design

## Getting Started

To create a new post, run:

```bash
hugo new posts/my-post.md
```

To preview the site locally, run:

```bash
hugo server -D
```

Then visit `http://localhost:1313` in your browser.
